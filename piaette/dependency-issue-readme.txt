För att få dependencies att fungera ordentligt kan man behöva gå in på properties->build path->libraries 
Öppna lwjgl.jar genom att trycka på pilen och dubbelklicka på Native. Tryck på workspace och leta upp den folder 
som matchar ditt operativsystem. 
Exempelvis piaette->dependencies->native->macosx
Tryck "OK". Beskåda hur problemen löser sig själva.
